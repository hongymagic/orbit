# Reference Mock

These files are the **original vanilla-React mock** that seeded the Orbit design system.

**They are reference-only.**

- Do not import from this directory.
- Do not run these files.
- The mock uses Babel-standalone + React 18 + a handcrafted `styles.css`. The production app under `src/` is Next.js 16 + Tailwind v4 + TypeScript.
- If a pattern is unclear in the production code, read the mock for original intent, then express it through the foundation (tokens, components, variations) — don't port literally.

See [DESIGN.md](../DESIGN.md) for the design system specification.
